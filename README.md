
### Install
`npm install`

### Run
`node ./bin/www`

Will be available at:
(http://localhost:3000/)[http://localhost:3000/]

## API

- `GET /api/topic`
Load all topics

- `GET /api/topic/:name/
Load all topic info, where `name` is name of topic

- `POST /api/topic`
Add new topic to list


- `DELETE /api/topic/:name/
Removes topic with name `name`
